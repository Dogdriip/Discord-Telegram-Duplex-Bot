# Discord-Telegram-Duplex-Bot

Duplex system for simple text message between Telegram and Discord

## How to use

1.  Install dependencies

    ```
    $ npm i
    ```

2.  Create `docker-compose.yml`.

    ```
    # See docker-compose-example.yml
    ```

3.  Run!

    ```
    $ docker-compose up -d
    ```
